# Memory Detail: UI Behavior Protocol

## Purpose
To ensure the AI assistant does not irritate the user by opening new windows or document panels that block the view of instructions.

## The Rule
1. NEVER open new windows, document panels, or separate views to display code or content.
2. ALWAYS display content inline within the chat response.
3. Do not use interactive pop-ups, modals, or external viewers.
4. Keep all code blocks and instructions in the main chat flow so the user can read everything in one place.

## Status
- Established: 2026-06-18
- Applies to: All AI assistants interacting with the user in chat interfaces.
