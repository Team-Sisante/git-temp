# Memory Detail: Code Block Copy Protocol

## Purpose
To ensure that code snippets shared in chat are immediately usable by the user without requiring manual editing or reconstruction.

## The Rule
1. NEVER use placeholder comments like "// ... rest of the command" or "// ... existing code" inside a code block.
2. ALWAYS provide the complete function or file content within the code block.
3. If the code is too long, break it into multiple complete functions/files rather than truncating.
4. Place all explanatory text, instructions, and comments OUTSIDE the code block.

## Status
- Established: 2026-06-18
- Applies to: All AI assistants sharing code content in chat.
