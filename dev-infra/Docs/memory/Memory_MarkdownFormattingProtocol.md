# Memory Detail: Markdown Formatting Protocol for Code Blocks

## Purpose
To ensure code snippets shared in chat don't break the page layout.

## The Rule
When sharing file contents containing backticks, use a code block with the bash language identifier (or any identifier) to wrap the content. Do not nest code fences inside each other.

## Status
- Established: 2026-06-18
- Applies to: All AI assistants sharing markdown/code content in chat.
